Positive Test Cases

Test Case ID: EC-344-TC-001
Title: Successful Checkout Information Entry
Test Type: Positive
Preconditions: User has a valid account with username standard_user and password secret_sauce.

Steps:
1:
  Action: Navigate to the Swag Labs application URL.
  Expected result: Application loads successfully with the login page displayed; username field, password field, and Login button are visible.

2:
  Action: Enter "standard_user" in the Username field.
  Expected result: Username field shows standard_user.

3:
  Action: Enter "secret_sauce" in the Password field.
  Expected result: Password field holds the value (masked).

4:
  Action: Click Login button.
  Expected result: User is redirected to the landing page /inventory.html; product list is visible.

5:
  Action: Click Add to Cart for the first product.
  Expected result: First product is added to the cart successfully.

6:
  Action: Click on the Cart icon on the top right-hand side.
  Expected result: Cart overview page appears, displaying the added product.

7:
  Action: Click Checkout button.
  Expected result: Checkout info page is displayed with First Name, Last Name, and Zip/Postal Code fields.

8:
  Action: Enter "Aradhana" in the First Name field.
  Expected result: First Name field shows Aradhana.

9:
  Action: Enter "Goyal" in the Last Name field.
  Expected result: Last Name field shows Goyal.

10:
  Action: Enter "11111" in the Zip/Postal Code field.
  Expected result: Zip/Postal Code field shows 11111.

11:
  Action: Click Continue button.
  Expected result: User proceeds to checkout overview page displaying product details correctly.

Expected Result: User successfully enters checkout information and proceeds to the checkout overview page displaying the correct product information.
