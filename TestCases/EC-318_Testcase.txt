Positive Test Cases:

Test Case ID: EC-318-TC-001
Title: Cart Cleanup and Continue Shopping - Positive Flow
Test Type: Positive
Preconditions: User is a authenticated user with username `standard_user` and password `secret_sauce`.
Steps:
1:
  Action: Navigate to the application URL (https://www.saucedemo.com/).
  Expected result: Login page loads; Username field, Password field, and Login button are visible.

2:
  Action: Enter 'standard_user' in the Username field.
  Expected result: Username field shows 'standard_user'.

3:
  Action: Enter 'secret_sauce' in the Password field.
  Expected result: Password field holds the value (masked).

4:
  Action: Click the Login button.
  Expected result: User is redirected to the inventory page; product list or inventory container is visible.

5:
  Action: Click the cart icon.
  Expected result: User navigates to the `/cart.html` page.

6:
  Action: Click Remove for each product in the cart.
  Expected result: Products are removed from the cart; the cart is empty.

7:
  Action: Click 'Continue Shopping' button.
  Expected result: User is redirected to the inventory page.

Expected Result: User successfully empties the cart and continues shopping back to the inventory page.
