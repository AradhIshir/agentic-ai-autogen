APP_URL: https://www.saucedemo.com/

Positive Test Case

Test Case ID: EC-334-TC-001
Title: Add Product to Cart Successfully
Test Type: Positive
Preconditions: User is an authenticated shopper with valid credentials.

Steps:

Step 1:
  Action: Navigate to the application URL.
  Expected result: The login page loads; username field, password field, and Login button are visible.

Step 2:
  Action: Enter "standard_user" in the Username field.
  Expected result: Username field shows standard_user.

Step 3:
  Action: Enter "secret_sauce" in the Password field.
  Expected result: Password field holds the value (masked).

Step 4:
  Action: Click the Login button.
  Expected result: User is redirected to the Products (Inventory) page; a list of products with their name, description, price, and Add to Cart button is visible.

Step 5:
  Action: Click the Add to Cart button for a product.
  Expected result: The selected product is added to the cart; the Add to Cart button changes to Remove, and the Cart icon badge updates with the correct item count.

Step 6:
  Action: Click on the Cart icon.
  Expected result: User is navigated to the Cart page showing the added product details (name, price, quantity).

Expected Result: User successfully adds a product to the cart and verifies it in the Cart page with correct details.
