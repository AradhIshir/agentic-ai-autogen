Positive Test Cases

Test Case ID: EC-343-TC-001
Title: Successfully Sort Products by Name A to Z
Test Type: Positive
Preconditions: User is already registered on the app
Steps:
1:
Action: Navigate to the application URL.
Expected result: Login page loads; username field, password field, and Login button are visible.

2:
Action: Enter "standard_user" in the Username field.
Expected result: Username field shows standard_user.

3:
Action: Enter "secret_sauce" in the Password field.
Expected result: Password field holds the value (masked).

4:
Action: Click the Login button.
Expected result: User is redirected to the inventory page.

5:
Action: Verify that the inventory page displays a sort dropdown with options.
Expected result: Sort dropdown is visible with options: Name (A to Z), Name (Z to A), Price (low to high), Price (high to low).

6:
Action: Select "Name (A to Z)" from the sort dropdown.
Expected result: Product list is re-ordered in ascending order by name.

Expected Result: User successfully sorts products by name A to Z.
